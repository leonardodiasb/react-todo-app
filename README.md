# react-todo-app

 - This is a clone of a To-Do webapp created using React. You can add items to a list, checkmark and delete them.

## Built With

- HTML/CSS
- Javascript
- React
- NPM

## Getting Started

### Install

- `git clone https://github.com/leonardodiasb/react-todo-app.git`
- `cd react-todo-app`
- `npm install`
- `npm start`
